Easy Definitions
PC:Easy TC:Very Low
A pair of linear equations in two variables that has no solution is called:
  • A A dependent pair
  • B An inconsistent pair
  • C A consistent pair
  • D An equivalent pair
Explanation
A pair of linear equations with no solution is called an inconsistent pair. Graphically, such lines are parallel and never meet. A consistent pair has at least one solution, and a dependent pair has infinitely many solutions.
Tests recall of the fundamental definition from Section 3.2. Addresses common confusion between "inconsistent" and "dependent".

MCQ
Easy Ratio Condition
PC:Easy TC:Low
For the pair of equations a₁x + b₁y + c₁ = 0 and a₂x + b₂y + c₂ = 0, which condition guarantees that the lines are coincident?
  • A a₁/a₂ ≠ b₁/b₂
  • B a₁/a₂ = b₁/b₂ ≠ c₁/c₂
  • C a₁/a₂ = b₁/b₂ = c₁/c₂
  • D b₁/b₂ = c₁/c₂ ≠ a₁/a₂
Explanation
Coincident lines represent a dependent pair with infinitely many solutions. This happens when a₁/a₂ = b₁/b₂ = c₁/c₂. Option B gives parallel lines (inconsistent). Option A gives intersecting lines (unique solution).
Tests the three ratio conditions from Table 3.1. Misconception: students often confuse the condition for coincident vs parallel lines since both have a₁/a₂ = b₁/b₂.

MCQ
Medium Ratio Comparison
PC:Medium TC:Low
Consider the pair of equations: 4x – 6y + 10 = 0 and 6x – 9y + 15 = 0. What is the graphical nature of these lines?
  • A Intersecting at exactly one point
  • B Coincident (same line)
  • C Parallel to each other
  • D Perpendicular to each other
Explanation
a₁/a₂ = 4/6 = 2/3  |  b₁/b₂ = –6/–9 = 2/3  |  c₁/c₂ = 10/15 = 2/3
Since a₁/a₂ = b₁/b₂ = c₁/c₂ = 2/3, the lines are coincident, giving infinitely many solutions. (Multiplying equation 1 by 3/2 gives equation 2.)
Tests ratio computation and identification of dependent pair. Students often make arithmetic errors when simplifying c₁/c₂ with negative constants, so this question avoids that to focus on the core concept.

Medium Graphical Solution
PC:Medium TC:Medium
The graph below shows two lines. Based on the graph, which statement BEST describes the pair of linear equations?
  • A The pair is consistent with a unique solution
  • B The pair is dependent with infinitely many solutions
  • C The pair is inconsistent with no solution
  • D The pair has exactly two solutions
Explanation
Lines L₁: y = x + 1 and L₂: y = x – 1 have the same slope (1) but different y-intercepts. They are parallel and never intersect, making the pair inconsistent with no solution. Ratio check: a₁/a₂ = 1/1, b₁/b₂ = –1/–1, c₁/c₂ = –1/1, so a₁/a₂ = b₁/b₂ ≠ c₁/c₂ (parallel case).
Tests graphical interpretation of parallel lines as inconsistent pair. The diagram makes the concept visually clear. Misconception: students sometimes think two non-intersecting lines might still share a solution at "infinity".

MCQ
Medium Word Problem
PC:Medium TC:Medium
Ravi has some ₹10 notes and ₹20 notes. The total number of notes is 15, and the total amount is ₹220. If x = number of ₹10 notes and y = number of ₹20 notes, which pair of equations correctly models this situation?
  • A x + y = 220 & 10x + 20y = 15
  • B x + y = 15 & 10x + 20y = 220
  • C x + y = 15 & 20x + 10y = 220
  • D x – y = 15 & 10x + 20y = 220
Explanation
The total count of notes: x + y = 15. The total value: 10x + 20y = 220. Option C incorrectly swaps the denominations. Option D uses subtraction incorrectly. The correct pair is Option B. Verification: x = 8, y = 7 → 8 + 7 = 15 ✓ and 80 + 140 = 220 ✓.
Tests the ability to translate a word problem into a pair of linear equations — a core skill from Section 3.1 and Exercise 3.1 Q1. Distractors target common errors: swapping equations and swapping coefficients.

MCQ
Hard Consistency Check
PC:Hard TC:Medium
For what value of k will the pair 3x + ky = 12 and 6x + 8y = 24 have infinitely many solutions?
  • A k = 2
  • B k = 6
  • C k = 4
  • D k = 8
Explanation
For infinitely many solutions: a₁/a₂ = b₁/b₂ = c₁/c₂
a₁/a₂ = 3/6 = 1/2  |  c₁/c₂ = 12/24 = 1/2
So we need b₁/b₂ = k/8 = 1/2, giving k = 4.
With k = 4: equation 1 × 2 gives exactly equation 2 → coincident lines.
Tests reverse application — finding k for a given consistency condition. This is a harder concept from Exercise 3.1 Q3/Q4 area. Misconception: students may set up the ratio as 8/k instead of k/8.

MCQ
Hard Graphical Intersection
PC:Hard TC:High
The diagram below shows the graphs of two lines. What is the solution of the pair of linear equations they represent?
  • A x = 3, y = 2
  • B x = 2, y = 3
  • C x = 1, y = 2
  • D x = 0, y = 1
Explanation
The two lines L₁: y = x + 1 and L₂: y = 2x – 1 intersect at the point (2, 3), marked on the graph. Verification: y = 2+1 = 3 ✓ and y = 2(2)–1 = 3 ✓. So the solution is x = 2, y = 3. The point of intersection gives the unique solution.
Tests reading intersection coordinates from a graph — core skill from Section 3.2 and Example 1. Common error: students read (y, x) as (x, y), hence the distractor Option A.

MCQ
Hard Triangle & Lines
PC:Hard TC:High
Lines L₁: 2x + y – 6 = 0 and L₂: x – y – 3 = 0 intersect at a point P. Line L₂ also meets the x-axis at Q. What are the coordinates of Q?
  • A (0, –3)
  • B (0, 3)
  • C (–3, 0)
  • D (3, 0)
Explanation
To find where L₂: x – y – 3 = 0 meets the x-axis, set y = 0:
x – 0 – 3 = 0 → x = 3
So Q = (3, 0).
Note: Option A (0,–3) is the y-intercept of L₂, a common confusion. On the x-axis, y = 0, not x = 0.
Tests finding x-intercept of a line — key skill for Exercise 3.1 Q7 (triangle formed by two lines and x-axis). Misconception: students confuse x-intercept (y=0) with y-intercept (x=0).

Integer
Medium Perimeter Problem
PC:Medium TC:Medium
A rectangular plot has its length 5 m more than twice its width. If the half-perimeter of the plot is 50 m, what is the width of the plot (in metres)?
Answer:
Explanation
Let width = x, length = y.
Equation 1: y = 2x + 5 (length is 5 more than twice width)
Equation 2: x + y = 50 (half-perimeter = length + width)
Substituting: x + (2x + 5) = 50 → 3x = 45 → x = 15
Width = 15 m, Length = 35 m. Verify: 15 + 35 = 50 ✓
Tests translating a real-life perimeter situation into a linear equation pair and solving — mirrors Exercise 3.1 Q5. The integer answer (15) keeps computation clean.

Medium Graphical Solution
PC:Medium TC:Medium
The lines represented by x + 2y = 8 and 2x – y = 6 intersect at the point (a, b). What is the value of a + b?
Answer:
Explanation
From equation 1: x = 8 – 2y
Substitute in equation 2: 2(8–2y) – y = 6 → 16 – 4y – y = 6 → 5y = 10 → y = 2
x = 8 – 4 = 4. So intersection = (4, 2).
a + b = 4 + 2 = 6
Tests finding the intersection point of two non-parallel lines. The question asks for a+b rather than just the point, requiring students to complete the full solution. Useful as a bridge to Section 3.3.
